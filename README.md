 📝 To-Do List Web App

A simple and responsive To-Do List application built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks by adding, completing, and deleting tasks.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface
- Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

To-Do-List
│
├── index.html
├── style.css
├── script.js
└── README.md

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
